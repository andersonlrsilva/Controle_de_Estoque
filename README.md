Projeto de sistema ERP para controle de estoque


Este projeto tem como objetivo desenvolver um sistema ERP (Enterprise Resource Planning) para controle de estoque. 
O sistema permitirá que os usuários gerenciem o estoque de produtos, 
acompanhem as entradas e saídas, e gerem relatórios para análise,
entre outras funcionalidades como controle de Rh, controle de frotas e entregas,
rotinas para controle de produção.

O sistema será desenvolvido utilizando a linguagem de programação Python.
O banco de dados utilizado será o Mysql.

O sistema tera licença MIT, permitindo que outros desenvolvedores possam 
utilizar, modificar e distribuir o código-fonte do projeto de forma livre.

Atualizações futuras incluirão a implementação de funcionalidades adicionais,
que serão aunciadas antecipadamente, dentro do programa

O projeto está em constante desenvolvimento e melhorias serão feitas com base 
no feedback dos usuários e nas necessidades do mercado.

Algumas funcionalidades serão vendidas como módulos adicionais, permitindo que 
os usuários personalizem o sistema de acordo com suas necessidades específicas.

O sistema tera inicialmente uma interfaca basica, mas futuramente será implementada 
uma interface mais moderna e intuitiva para melhorar a experiência do usuário. 

O projeto será mantido e atualizado regularmente para garantir a segurança, 
estabilidade e desempenho do sistema.
